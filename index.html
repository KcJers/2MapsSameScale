2 Maps In Same Scale

This file: https://kcjers.github.io/2MapsSameScale/

<2025-02-02> Version 1.0: Access Here

This HTML web page displays two maps that can be synchronously zoomed in or out at the same scale/level.

  • Each map can be panned to desired locations for comparison.
  • Each map can be rotated, allowing the geographical locations and their shapes to be oriented similarly. A Heading indicator at the top-right corner shows the bearing.
  • For Windows PCs without a touch screen, rotation is done by holding Alt + Shift and dragging the cursor.
  • A Reset-To-North button resets the map to show North pointing to the top.
  • You can share the 2-map web page of your desired locations using the Share button to copy its web link (URL).
